Homeland

Descrição:

    Este projeto tem como objetivo desenvolver uma página web responsiva que apresenta as cidades natais de funcionários da TripleTen, aplicando práticas de HTML, CSS, BEM e design responsivo guiado pelo protótipo do Figma.
    O foco principal é construir um layout adaptável para diferentes resoluções de tela (320px, 768px e 1280px), utilizando breakpoints estratégicos e técnicas modernas de layout.


Funcionalidades:

    - Estrutura completa do projeto seguindo arquitetura de diretórios BEM.
    - Configuração do index.html, folhas de estilo e conexão da fonte externa “Inter”.
    - Layout completo para três larguras principais conforme o roteiro do Figma.
    - Design responsivo utilizando breakpoints em 544px e 1024px.
    - Adaptação visual detalhada para resoluções intermediárias.
    - Uso de técnicas como flexbox, grid, object-fit e unidades relativas (% e fr).
    - Otimização e compressão de imagens para melhor desempenho.


Tecnologias Utilizadas:

    - HTML5
    - CSS3 (Flexbox, Grid, Media Queries, BEM)
    - Git e GitHub
    - Figma para análise do layout
    - Fonte Inter (versão Web)


Objetivos de Aprendizado:

    - Aplicar boas práticas de organização de arquivos usando a metodologia BEM.
    - Desenvolver habilidades de design responsivo com breakpoints eficientes.
    - Aprender a interpretar layouts do Figma e convertê-los em código real.
    - Aprimorar o domínio de HTML e CSS, priorizando estilos limpos, escaláveis e consistentes.
    - Criar uma base sólida para projetos mais avançados que envolvam lógica e JavaScript.
